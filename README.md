# Como executar o projeto

## Back end
Pré-requisitos: Node

```bash
# clonar repositório
git clone https://github.com/SanzioWesley/PortalDeNoticias.git
# entrar na pasta do projeto curso-node
cd curso-node

# executar o projeto
node app
```
